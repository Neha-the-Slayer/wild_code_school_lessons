A list of essential Bash commands

  • cd < directory path >: Change directory. Use it to move from one folder to the other. Both relative and absolute paths can be provided
  • ls: List Storage. Shows all the visible directories and files
  • pushd < directory path >: Will move from current directory to the specified one but will kep the current directory in memory
  • popd: To be used after pushd is used. It brings you back to the directory you were in before using pushd
  • file < file or directory name>: Prints information about a file type (or directory)
  • locate < string >: Search for all files/directory that contains the provided string in their name
  • sudo updatedb: To updated the database for use of locate command
  • which < command > : It checks if a command is installed, printing where it is installed
  • history: It lists the last 1000 unique commands issued
  • whatis < command >: It gives you a very short explaination of what a command does
  • apropos < string >: Return a list of all commands that have something to do with the string provided. It's a way to tell the system what you want to do and get a list of related commands
  • man < command >: It shows the manual page for the chosen command providing detail information
  • mkdir < dirName > < dirName > ...: Make a directory
  • touch < fileName > < fileName > ... < fileName >: Creates/Update a file
  • cp < /fileName > < new fileName >: Will make a copy of the file in the directory you are working to
  • mv < /fileName or directory > < new fileName or directory >: Change the name of a file/directory
  • mv < /fileName or directory > < directory >: Move the file or the directory to a different directory
  • rm < /fileName >: Removes the file
  • rmdir < dirName >: Removes a directory only if it doesn't have files in it
  • cat < /fileName >: Prints the content of the file
  • more < /fileName >: It pages through a bunch of text in the bash. Sort of devide the content into pages and let your go through them. Press "q" to exit
  • less < /fileName >: A bette version of more. Can use arrow keys and go down/up one line at a time. You can search for text etc..
  • nano < /fileName >: Integrated text editor in most linux distribution. If the file doesn't exist it is created
  • sudo < command >: It will allow to run commands as administrator
  • su - < userName >: This command changes you to another user. The dash after the command will make you also move to the user home directory. With no dash, you will stay in your home directory
  • users: Will show you users logged in the system
  • chmod < code > < /fileName >: Change the permission for files and directories
  • watch < command >: Will run a program every 2 seconds and update the bash if anything chages
  • killall < programName >: It kills all the process related to the program

Additional Information

List Storage(ls) flags

  1. ls -l: Lists all visible directories with access rights in a list form.
  2. ls -a: Shows all visible and hidden directories
  3. ls -al: Combination of -a adn -l flags
  4. ll: Same as ls -l

Locate

Locate works based on an internal database that gets updated once a day. If you are looking for something that was created recently, locate might not work. You can manually updated the database.

Remove(rm) flags

  1. rm *: Removes all the file from a directory
  2. rm work*: Removes all the file with a name that starts with work
  3. rm -r: To remove a directory

Concatenate(cat) Operators and Flags

  1. cat > < /fileName >: Will let you type in the bash and whatever you type will be saved in the specified file. If the file has text, it will be replace. Use ctrl+d to finsih
  2. cat >> < /fileName >: Will let you type in the bash and appends it at the end of fileName
  3. cat < /fileName1 > < /fileName2 >: Takes the content from fileName2 and appends it at the end of fileName1
  4. rm -r: To remove a directory

Redirection

  1. / > : redirect a text output. For example: ls -al / > file.txt the list storage will be saved in file.txt
  2. cat >> < /fileName >: Will let you type in the bash and appends it at the end of fileName
  3. cat < /fileName1 > < /fileName2 >: Takes the content from fileName2 and appends it at the end of fileName1
  4. rm -r: To remove a directory

Piping ( | )

Taking the output of a command and using that as input for another command. For Example: history | less. With this command you will be able to page through the list of commands you ran.
